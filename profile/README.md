## 📌 How to Contribute

1. Fork or clone the repository
2. Create your project inside the correct folder
3. Use clear naming:
     yourname-project-name
4. Commit regularly
5. Push your work

For collaborative projects:
- Create a branch
- Open a Pull Request
- Request review
